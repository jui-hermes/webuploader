---

http://fex.baidu.com/webuploader

## Install

```
$ spm install webuploader --save
```

## Usage

```js
var WebUploader = require('webuploader');
var uploader = new WebUploader.Uploader({
    swf: 'path_of_swf/Uploader.swf'

    // 其他配置项
});
```
